# Ecos - Frontend

## Tecnologias

[![Tecnologias](https://skillicons.dev/icons?i=react,ts,tailwind,vite,eslint,prettier)](https://skillicons.dev)

## Librerias principales

- React Router
- Axios
- React Hook Form
- Zod
- TanStack Query
- Vitest
- Testing Library

## Requisitos previos

- Node.js
- npm

## Variables de entorno

Tomar como referencia el archivo `.env.example`.

```env
VITE_API_URL=url-a-la-api
```

## Ejecutar localmente

```bash
cd frontend
npm install
npm run dev
```

## Scripts

```bash
npm run build
npm run lint
npm test -- --run
npm run coverage
```

## Estructura

El proyecto organiza las vistas y componentes por dominio funcional, con carpetas para autenticacion, home, eventos, perfiles y componentes compartidos.
